# training-eval-overlap
Quantificar a sobreposição entre o corpus Carolina (pré-treino de um LLM em português baseado no Qwen) e as instâncias de avaliação do benchmark PoetaV2. O objetivo é verificar se perguntas, trechos ou respostas aparecem total ou parcialmente no treino, o que poderia inflar artificialmente os resultados do benchmark.
